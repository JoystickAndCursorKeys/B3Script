# B3Script

Plugin for embedding "Basic" as a scripting language into HTML

![](img/feature1.png)

---

B3Script stands for Basically Browser Basic for WWW (W3) 

Or if you prefer B3, B3 Script or  B3 Browser Basic :)

Short: B3 or B3Script
